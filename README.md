**Hiraku Kurafuto** is a next-generation voxel sandbox game that puts developers, modders, and players first. Built from the ground up for infinite scale and absolute freedom, this engine is a blank canvas. We provide the performance and the API; you build the worlds.

### ‚ö° The Engine

* **Infinite Voxel Worlds:** High-performance, endlessly generating, and fully destructible terrain.
* **Deep Engine Access:** A robust, unrestricted API designed to let plugins and mods alter core mechanics without friction.
* **Radical Transparency:** 100% open-source, forever. Development is driven by community consensus and active player feedback.
